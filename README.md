# Telegram Car Store Bot

Профессиональный Telegram-бот для автосалона, построенный на **Aiogram 3.x** и **SQLAlchemy 2.0**.

## Возможности

### Для пользователей:
- **Каталог**: Просмотр автомобилей по категориям.
- **Поиск**: Удобный поиск по марке и модели.
- **Карточки авто**: Подробная информация с фото, ценой и описанием.
- **Избранное**: Возможность сохранять понравившиеся машины.
- **Заявки**: Форма оформления заявки на покупку.
- **Контакты**: Связь с менеджером и адрес салона.

### Для администраторов:
- **Админ-панель**: Управление всем каталогом.
- **Управление авто**: Добавление, редактирование и удаление машин прямо через бот.
- **Уведомления**: Мгновенные уведомления о новых заявках от клиентов.
- **Просмотр заявок**: История всех входящих запросов.

## Технологии
- **Python 3.11+**
- **Aiogram 3.x** (Асинхронный фреймворк для Telegram ботов)
- **SQLAlchemy 2.0** (ORM для работы с БД)
- **SQLite** (Легкая и надежная база данных)
- **Pydantic Settings** (Управление конфигурацией)


   ```bash
   python init_data.py
   ```

4. Запустите бота:
   ```bash
   python main.py
   ```

## Структура проекта
- `data/`: Файлы базы данных.
- `handlers/`: Обработчики команд и сообщений.
- `keyboards/`: Скрипты для создания кнопок.
- `models/`: Описание таблиц БД и операции с ними.
- `states/`: Состояния FSM (конечный автомат).
- `utils/`: Вспомогательные скрипты и конфигурация.
